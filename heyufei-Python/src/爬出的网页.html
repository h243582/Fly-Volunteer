　　北京城市学院是经国家教育部批准成立的一所综合性普通高校，具有颁发国家承认的研究生、本科学历学位资格，并举办中职、高职专科教育。学校创建于1984年，是新中国第一所实行公有民办体制的新型高校，以务实、创新而著称，被誉为中国高等教育改革的一面旗帜。[详细]

专业推荐指数Top10 说明 
专业推荐说明：

1. 由普通本专科学生根据本校各专业办学情况，推荐优势专业或特色专业（通过实名注册的学生才具有推荐资格）。

2. 推荐结果仅代表学生对本校相关专业的推荐意见，不代表官方的专业排名，不同学校间不宜做直接比较。

3. 推荐的专业范围包括所有在校学生所学专业和上年度新增的招生专业。

4. 系统自动显示推荐票数较多且推荐指数较高的部分专业，投票人数较少的高校暂时不显示或只显示少量专业。
